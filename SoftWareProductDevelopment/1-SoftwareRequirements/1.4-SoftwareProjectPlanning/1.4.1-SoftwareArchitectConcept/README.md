### Концептуальний опис архітектури програмного продукту

**Архітектурний тип:** Mixed Application (RWA/MA)

| Рівень | Компонент | Особливості реалізації |
| :--- | :--- | :--- |
| **PL (Presentation Level)** | Web-навігатор | **Апаратний компонент:** Настільний ПК, Смартфон. **Програмний:** Chrome, Firefox, Safari. **Технологія:** React/Vue (JS-фреймворк). |
| **BL (Business Level)** | Сервер застосунків | **Апаратний компонент:** Сервер. **Програмний:** OS Linux. **Технологія:** Node.js або Spring Boot (обробка даних, генерація резюме). |
| **AL (Access Level)** | База даних | **Апаратний компонент:** Сервер. **Програмний:** OS Linux. **СКБД:** **PostgreSQL (Реляційна модель)**. |
