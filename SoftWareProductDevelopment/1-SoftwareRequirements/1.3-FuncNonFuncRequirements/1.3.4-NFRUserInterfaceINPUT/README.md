### Опис вимог до INPUT-потоків інтерфейсу користувача

Таблиця 1.5 - Приклад оформлення результатів аналізу засобів INPUT-потоків

| FR | NFR | Засіб INPUT-потоку | Особливості використання |
| :--- | :--- | :--- | :--- |
| FR1.1 | **NFR2.1** | Стандартна комп'ютерна клавіатура | Введення ключових слів та використання клавіші **Enter** для активації пошуку (як альтернатива натисканню кнопки). |
| FR1.1 | **NFR2.2** | 2/3-кнопочний маніпулятор типу "миша" | Натискання кнопки **"Пошук"**, клік на прапорцевій кнопці **"Розширений пошук"**. |
| FR1.1 | **NFR2.3** | Сенсорний екран (для мобільної версії) | Використання екранної клавіатури для введення тексту, дотик (tap) для активації пошуку та фільтрів. |
