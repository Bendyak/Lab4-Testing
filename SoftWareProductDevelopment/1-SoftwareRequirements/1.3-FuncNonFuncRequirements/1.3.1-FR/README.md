### Опис функціональних вимог

Таблиця 1.2 - Приклад опису функціональних вимог, які враховують умови стандарту

| FR | Опис функціональної вимоги |
| :--- | :--- |
| **FR1** | **Пошук наукових ресурсів** |
| FR1.1 | Користувач вводить ключові слова для пошуку наукових ресурсів. |
| FR1.2 | ПП знаходить перелік актуальних наукових статей. |
| FR1.3 | ПП генерує зрозумілі резюме для кожної знайденої статті. |
| FR1.4 | ПП надає користувачу зрозумілі резюме для зменшення часу розуміння інформації. |
| **FR2** | **Пошук житла** |
| FR2.1 | Користувач вводить параметри фільтрації, включаючи "безпечний район" та "наявність робочого простору". |
| FR2.2 | ПП відображає список актуальних пропозицій житла, що відповідають фільтрам. |
| FR2.3 | ПП забезпечує безпечні та комфортні умови для роботи. |
