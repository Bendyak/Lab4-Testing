### 2.3.2 Проблемний аналіз існуючих програмних продуктів

| Потреба | Продукт / сайт | Тип | Короткий опис | Недоліки |
|---|---|---|---|---|
| Житло | OLX (olx.ua) | Пропрієтарна | База оголошень по оренді | Відсутня верифікація; ризик шахрайства |
| Житло/короткостроково | Booking (booking.com) | Пропрієтарна | Зручна для тимчасової оренди | Не оптимізована для довгострокових студентських договорів |
