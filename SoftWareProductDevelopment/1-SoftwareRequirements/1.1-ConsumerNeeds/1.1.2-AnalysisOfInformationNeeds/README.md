### 2.3.1 Аналіз інформаційних потреб

| Інформаційна потреба | Що саме потрібно | Джерела інформації | Періодичність оновлення | Коментар |
|---|---|---|---:|---|
| Їжа | Списки кафе, меню, доставка, інформація про харчову цінність | Сервіси доставки, API ресторанів, локальні каталоги | Щодня/щотижня | Фільтри: дієта, ціна |
| Житло | Оголошення оренди, перевірені контакти, умови та ціна | OLX, портали оренди, локальні групи | Щодня | Потрібна верифікація оголошень |
| Професійне спілкування | Афіші подій, форуми, контакти колег | Університетські портали, соцмережі, платформи подій | Щомісяця | Підписка на теми/регіон |
| Наукова діяльність | Публікації, гранти, конференції, лабораторії | Google Scholar, ResearchGate, сайти конференцій | В міру публікацій | Пріоритизація за темами досліджень |

**Пояснення (для кожного незадоволення):**
- Для їжі: користувачеві важко швидко знайти здорові й доступні опції поруч із кампусом.  
- Для житла: багато оголошень без верифікації — низька довіра.  
- Для спілкування: інформація розкидана, складно відслідковувати релевантні події.  
- Для науки: немає єдиної цільової стрічки публікацій/грантів для конкретних тем.
