## 1.2.3 Use Case Diagram  

На діаграмі варіантів використання зображені основні актори та їх взаємодія з системою:

- **Студент / Дослідник** — здійснює пошук житла, їжі та наукових ресурсів.  
- **Орендодавець** — розміщує оголошення.  
- **Адміністратор** — підтверджує та перевіряє дані.  
- **Зовнішні сервіси** (OLX, служби доставки, наукові бази) — забезпечують додаткову інформацію.  

Ключові прецеденти:  
- Пошук житла  
- Пошук харчування  
- Пошук наукових ресурсів  
- Фільтрація результатів  
- Перевірка оголошень  
- Сповіщення користувачів
