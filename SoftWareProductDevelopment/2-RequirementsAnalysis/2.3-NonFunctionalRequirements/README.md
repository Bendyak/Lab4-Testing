# 2.3 Non-Functional Requirements (Нефункціональні вимоги)

## Продуктивність:
- Система повинна обробляти запити користувачів не довше ніж за 3 секунди.

## Надійність:
- Відновлення після збою має займати не більше 10 хвилин.

## Безпека:
- Усі запити передаються через HTTPS.
- Дані користувачів зберігаються у зашифрованому вигляді.

## Зручність використання:
- Інтерфейс інтуїтивно зрозумілий і локалізований українською мовою.

## Сумісність:
- Підтримка сучасних браузерів: Chrome, Firefox, Edge, Safari.
