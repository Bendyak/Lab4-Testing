# 2.1 Functional Requirements (Функціональні вимоги)

## Основні функціональні можливості системи:
- Реєстрація користувачів (студентів, дослідників, адміністраторів).
- Авторизація та розмежування доступу відповідно до ролей.
- Пошук житла, харчування, наукових ресурсів через інтеграцію зі сторонніми сервісами.
- Збереження персональних налаштувань користувача.
- Формування звітів про використання ресурсів.

## Примітка:
Усі функції повинні бути доступні через веб-інтерфейс із захищеним з’єднанням (HTTPS).
