# 2.2 Use Cases (Прецеденти використання)

## Основні сценарії:
1. **Пошук житла**
   - Користувач вводить параметри пошуку (місто, ціна, тип житла)
   - Система відображає доступні варіанти.

2. **Пошук варіантів харчування**
   - Користувач задає параметри (тип кухні, ціна, відстань)
   - Система показує релевантні результати.

3. **Пошук наукових ресурсів**
   - Користувач вводить ключові слова.
   - Система отримує результати з Google Scholar, ResearchGate тощо.

4. **Адміністрування**
   - Адміністратор може переглядати активність користувачів і змінювати налаштування системи.
